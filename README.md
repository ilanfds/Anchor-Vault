# Anchor Vault

A learning project for the Anchor Framework on Solana blockchain.

## About

This project explores how to build smart contracts using Anchor, a powerful framework for Solana development.

## Getting Started

```bash
# Install dependencies
yarn install

# Build the program
anchor build

# Run tests
anchor test
```
